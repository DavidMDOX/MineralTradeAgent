# 交付说明

## 定位
本项目是“矿产贸易企业智能经营平台”的演示与交付基础版，覆盖市场、采购、生产、销售和多 Agent 分析。

## 数据说明
当前数据为贴近真实经营场景的虚拟样本，不代表真实交易或真实矿山生产数据。

## 适合直接演示的页面
- `/dashboard`
- `/market`
- `/procurement`
- `/production`
- `/sales`
- `/agents`

## 后续可继续扩展
- PostgreSQL / MySQL
- 真正的供应商、库存、ERP、CRM 对接
- 定时任务与消息提醒
- 更细颗粒度的权限体系
- 更强的多 Agent 协同与工具调用
